My Recent Projects

My Mood App

This cross-platform mobile application is designed to enhance emotional well-being by combining AI-powered mood detection with personalized exercise recommendations. Users can either let the AI analyze their emotions in real-time through live face analysis or manually select their mood via emojis, after which the app suggests exercises to improve emotional balance. Beyond automated recommendations, users can also contribute their own personalized tips and exercises to support others in the community, fostering a collaborative experience. The app integrates secure payment solutions, allowing safe credit card storage, and uses RevenueCat to manage in-app purchases for premium features across both Google Play and the App Store. Built with Flutter, connected to RESTful APIs, and designed with a focus on scalability, security, and user experience, the app represents a blend of AI innovation, wellness, and modern app monetization.

Shamsouna E-Commerce App Store

A full-featured e-commerce mobile application developed using Flutter, enabling users to browse and purchase electronic products and request technical repair services. Features include secure authentication, interactive location selection, payment processing (Tap Payments Gateway), order tracking, and a separate Captain app for service providers to manage requests. Inspired by the bidding model used in apps like InDrive.

Flutter Google Maps REST APIs Tap Payments Clean Architecture

Bookly App

A book discovery app developed using Flutter that allows users to browse and download free books. It includes secure login, registration, and password reset with Firebase Authentication, and real-time filtering for book categories and keywords. The app follows Clean Architecture principles, SOLID principles, and MVVM for a scalable and maintainable codebase, optimized for mobile, tablet, and web platforms with responsive design.

Flutter Dart Firebase Auth REST APIs Bloc MVVM
Watch Demo Video

Fruits E-Commerce App

A cross-platform mobile app for fruit shopping developed using Flutter. It features secure email/password authentication with Google and Facebook login support, PayPal integration for online payments, product search and filtering, order tracking, and profile management. The app supports light and dark themes, multilingual support (Arabic & English), and applies MVVM architecture and Bloc for state management, with a responsive UI for consistent experience across devices.

Flutter Dart Firebase PayPal SDK Bloc MVVM
